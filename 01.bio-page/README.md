# Project 01: Bio Page (Portfolio)

A minimalist, responsive personal portfolio page designed to showcase my background, transition into tech, current studies, and hobbies. 

This project was built as part of my journey in **CS50’s Web Programming with Python and JavaScript (CS50W)**.

---

## Visual Identity & Aesthetic

The page features a carefully curated, modern editorial design focusing on clean typography and subtle, harmonious tones:
* **Palette:** A soft balance of calming sage greens (`#3d5a46`, `#cad2c5`) paired with delicate dusty rose accents (`#f5e6e8`, `#d6b2b8`).
* **Typography:** `Playfair Display` (serif) for elegant headings, paired with `Inter` (sans-serif) for high-readability body text.

---

## Key Features

* **Hero Section:** Features a stylized circular profile picture with custom borders/shadows and a brief intro.
* **Responsive Layout:** Built using Bootstrap 5's grid system, ensuring a seamless experience across mobile, tablet, and desktop screens.
* **Interactive `<details>` Element:** Custom-styled to show extra personal details with an elegant cursor trigger and soft background colors.
* **Polished Academic/Studies Table:** Fully customized Bootstrap table featuring delicate hover states, clean custom column widths, and styled pills (`.tag-clean`).
* **Custom Buttons:** Hand-styled call-to-action buttons in the footer that transition smoothly on hover.

---

## Tech Stack & Concepts Applied

* **HTML5:** Semantic structuring (using elements like `<details>`, `<summary>`, `<footer>`, and tables).
* **CSS3:** Custom variables, transition effects, shadow properties, and layout overrides.
* **Bootstrap 5:** Grid system, typography helpers, and utility classes customized through overriding CSS.

---

## File Structure

* `index.html` — The main structure of the bio page.
* `style.css` — Custom stylesheet overriding Bootstrap defaults with a custom color palette and refined details.
* `photo_bio.jpg` — Profile picture.
